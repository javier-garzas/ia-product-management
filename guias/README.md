# Guías

Guías de **Javier Garzás — 233 Academy** para **leer y aprender** (pensadas para personas, a diferencia de las [skills](../README.md), pensadas para agentes de IA).

| Guía | De qué va | Licencia |
|------|-----------|----------|
| [Guía BDD y Gherkin](guia-bdd-gherkin.md) | Guía de supervivencia de **BDD, Gherkin y Cucumber**: el proceso de fuera a dentro, los tres amigos, el lenguaje común, Features y Scenarios (Given/When/Then), y buenas prácticas. Complementa a las skills `crear-historias-de-usuario` y `crear-spec-sdd`. | CC BY-NC-SA 4.0 |
| [Guía de OKRs](guia-okrs.md) | Guía completa de **OKRs**: objetivos y resultados clave, cómo crearlos y puntuarlos, MOKRs/CFRs, la Métrica de la Estrella del Norte, antipatrones y más. Es también la base de conocimiento de la skill [`crear-y-revisar-okrs`](../skills/crear-y-revisar-okrs/). | CC BY-NC-ND 4.0 |
